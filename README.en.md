# AvZ VSCode Extension

A VSCode extension for the _Plants vs. Zombies_ TAS framework - [AsmVsZombies](https://github.com/vector-wlc/AsmVsZombies).

## Usage

|Features|Operations|
|:-:|:-:|
|View Tutorial|Press <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd> and type command `AvZ: Open Tutorial`|
|Set AvZ Installation Directory|Press <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd> and type command `AvZ: Set AvZ Dir`|
|Show AvZ Info|Press <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd> and type command `AvZ: Show AvZ Info`|
|Update/Rollback AvZ Version|Press <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd> and type command `AvZ: Update AvZ`|
|Download AvZ Extension|Press <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd> and type command `AvZ: Download AvZ Extension`|
|Run Script|Right-click the script code and select `AvZ: Run Script` in the menu / Click the _Run_ button in the top right of the editor|
