# AvZ VSCode Extension

用于 植物大战僵尸 键控框架 [AsmVsZombies](https://github.com/vector-wlc/AsmVsZombies) 的 VSCode 扩展。

## 用法

|功能|操作|
|:-:|:-:|
|查看使用教程|按下 `Ctrl + Shift + P` 后输入命令 `AvZ: Open Tutorial`|
|设置 AvZ 安装目录|按下 `Ctrl + Shift + P` 后输入命令 `AvZ: Set AvZ Dir`|
|显示 AvZ 版本信息|按下 `Ctrl + Shift + P` 后输入命令 `AvZ: Show AvZ Info`|
|进行 AvZ 版本的更新与回退|按下 `Ctrl + Shift + P` 后输入命令 `AvZ: Update AvZ`|
|下载 AvZ 插件|按下 `Ctrl + Shift + P` 后输入命令 `AvZ: Download AvZ Extension`|
|运行脚本|点击鼠标右键后选择菜单中的 `AvZ: Run Script`，或者点击编辑器右上角的运行按钮|
