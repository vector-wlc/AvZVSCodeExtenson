# AvZ README

VSCode Extension For PvZ TAS Framework : AsmVsZombies

## Command

* 右键选择 `AvZ : Run Script` 运行脚本

* 快捷键 `Ctrl + Shift + P` 调出命令面板后输入 `AvZ : Update` 进行 AvZ 版本的更新与退回
